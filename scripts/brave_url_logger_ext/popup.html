URL Logger - Installed Successfully

Thank you for installing URL Logger! This extension lets you quickly save URLs to a log file without dealing with popups.

Ways to Use URL Logger:

Keyboard Shortcuts

Ctrl + Shift + L (Windows/Linux) or ⌘ + Shift + L (Mac): Log the current URL

Ctrl + Shift + K (Windows/Linux) or ⌘ + Shift + K (Mac): Download the URL log file

Context Menu (Right-Click)

Right-click anywhere on a page and select:

"Log this URL" - to save the current page's URL

"Download URL log" - to save all logged URLs to a file

Toolbar Button

Click the URL Logger icon in your browser toolbar to instantly log the current URL

Each logged URL includes a timestamp and the page title. You can view your log file by using the download option.

Note: You can change the keyboard shortcuts by going to brave://extensions/shortcuts
